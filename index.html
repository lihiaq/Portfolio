Run Telem

Introduction

Run - Telem is an educational game generator designed to help students practice to identify different items. The game generator was created as part of a graduation project in H.I.T college. H.I.T maintain portal that contains dozens of games, Run Telem is chosen to be one of them.

During the project we had to cope with practical, authentic challenges posed by actual client. The project included: Needs Analysis, Proposing Solution/s, Design & Development of the solution and finally Implementation, Testing & Evaluation.
